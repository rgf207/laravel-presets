# laravel-presets
Add Set of Presets for Laravel Install

## Installation
edit composer.json add:

```json
"repositories": [
        {
            "type": "vcs",
            "url": "git@github.com:robbfountain/laravel-presets.git"
        }
    ],
    "require": {
    "onethirtyone/laravel-preset": "*"
```
